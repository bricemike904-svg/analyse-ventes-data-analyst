# Analyse des ventes d'un commerce local

## Contexte
Un petit commerce souhaite comprendre ses ventes afin d'identifier les produits les plus rentables et les jours les plus performants.

## Objectifs
- Nettoyer un jeu de données réel contenant des erreurs.
- Analyser les ventes par produit et par jour.
- Calculer le chiffre d'affaires.
- Fournir des recommandations business

## Outils
- Python
- Pandas
- Matplotlib

## Méthodologie
1. Exploration et nettoyage des données.
2. Correction des types et suppression des valeurs invalides.
3. Création du chiffre d'affaires
4. Analyses statistiques
5. Visualisations

## Résultats clés
- Le riz génère la majorité du chiffre d'affaires
- Le dimanche est le jour le plus performant
- Le riz représente plus de 50% du chiffre d'affaires total

## Recommandations
- Renforcer les stocks de riz les jours à forte demande.
- Mettre en place des promotions ciblées le dimanche.
